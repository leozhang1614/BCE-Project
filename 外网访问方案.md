# BCE 项目外网访问方案

## 问题
磊哥在外网，无法访问内网地址 `192.168.31.187:3000`

## 方案 1：GitHub Pages（推荐）⭐

**步骤：**
```bash
# 1. 推送到 GitHub
cd /Users/ai/.openclaw/workspace-jiangxin/匠心工作区/代码仓库/BCE-Project
git add public/index.html
git commit -m "Add static board for external access"
git push origin master

# 2. 启用 GitHub Pages
# Settings → Pages → Source: main branch /public folder
```

**访问地址：**
```
https://leozhang1614.github.io/BCE-Project/
```

**优点：**
- ✅ 免费
- ✅ 永久可用
- ✅ HTTPS 安全
- ✅ 自动 CDN

---

## 方案 2：Ngrok 内网穿透

**步骤：**
```bash
# 1. 安装 ngrok
brew install ngrok

# 2. 启动穿透
ngrok http 3000
```

**访问地址：**
```
https://xxx-xxx-xxx.ngrok.io
```

**优点：**
- ✅ 实时同步内网数据
- ✅ 无需部署

**缺点：**
- ❌ 需要持续运行
- ❌ 免费版有访问限制
