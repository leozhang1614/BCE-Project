## 匠心角色说明

**角色定位：** 技术负责人 / 执行者 / 开发者

**核心职责：**
- ✅ 接收任务
- ✅ 确认任务（30 分钟内）
- ✅ 执行任务开发
- ✅ 每 30 分钟反馈进度
- ✅ 提交验收

**权限范围：**
- ✅ 确认任务
- ✅ 更新进度
- ✅ 提交验收
- ✅ 申请转交任务
- ✅ 申请延期

### 任务通知

**收到任务分配通知（飞书）：**

```

任务：北斗智投系统 - 智能决策模块
描述：负责三维分析建议（日/周/月维度）
优先级：P0
截止时间：2026-03-24 17:00
负责人：司库

请在 30 分钟内确认任务！

[确认接收] [需要沟通]
```

## 任务确认

### 确认任务（30 分钟内）

**API 调用：**

```javascript
// 确认接收任务
const confirmTask = {
  operator: '匠心',
  comment: '任务已确认，立即开始执行。'
};

const response = await fetch('http://localhost:3000/api/bce/tasks/task-123/confirm', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify(confirmTask)
});

const result = await response.json();
console.log('✅ 任务确认成功:', result);
```

**成功响应：**

```json
{
  "success": true,
  "message": "任务确认成功",
  "data": {
    "taskId": "task-123",
    "status": "executing",
    "confirmedAt": "2026-03-23T10:15:00Z",
    "nextUpdateDue": "2026-03-23T10:45:00Z"
  }
}
```

---

### 需要沟通（不确认）

**如果有疑问，点击"需要沟通"：**

```javascript
// 申请沟通
const needComm = {
  operator: '匠心',
  action: 'needComm',
  reason: '需要确认三维分析的具体维度和算法要求。'
};

const response = await fetch('http://localhost:3000/api/bce/tasks/task-123/confirm', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify(needComm)
});
```

**系统会暂停计时，等待管理者介入。**

## 提交验收

### 开发完成，提交验收

**当任务完成后，提交验收：**

**方式一：飞书卡片**

点击"提交验收"，填写：

```
交付物：
  - 智能决策模块代码
  - 三维分析算法文档
  - API 接口文档
  - 单元测试报告

自测报告：
  - 功能测试：通过
  - 性能测试：通过
  - 边界测试：通过

备注：请验收
```

---

**方式二：API 提交**

```javascript
// 提交验收
const submitAcceptance = {
  operator: '匠心',
  deliverables: [
    '智能决策模块代码',
    '三维分析算法文档',
    'API 接口文档',
    '单元测试报告'
  ],
  selfTestReport: `
自测报告：
- 功能测试：通过
- 性能测试：通过
- 边界测试：通过
  `,
  comment: '开发完成，请验收。'
};

const response = await fetch('http://localhost:3000/api/bce/tasks/task-123/submit', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify(submitAcceptance)
});

const result = await response.json();
console.log('✅ 验收提交成功:', result);
```

**成功响应：**

```json
{
  "success": true,
  "message": "验收提交成功，已流转到待验收节点",
  "data": {
    "taskId": "task-123",
    "status": "pending_acceptance",
    "submittedAt": "2026-03-23T16:00:00Z",
    "nextStep": "等待司库验收"
  }
}
```

---

### 验收通过

**司库验收通过后，系统自动流转到"待审核"节点，通知执矩审核。**

---

### 验收不通过

**如果验收不通过，系统自动回退到"开发中"节点：**

```
🔔 验收不通过通知

任务：北斗智投系统 - 智能决策模块
验收人：司库
原因：
  1. 缺少性能测试报告
  2. 文档格式不规范

请重新开发并提交验收。
```

**需要重新开发，重新提交验收。**

## 常见问题

### Q1: 忘记确认任务怎么办？

**系统会在 30 分钟内多次提醒，如果超时未确认：**
- 30 分钟：L1 提醒（通知执行者）
- 60 分钟：L2 警告（通知管理者）
- 120 分钟：L3 严重（通知 CEO）

**解决方案：** 立即确认任务，或申请"稍后确认"。

---

### Q2: 进度更新失败怎么办？

**可能原因：**
1. BCE 服务未启动
2. API 地址错误
3. 参数格式错误

**解决方案：**
```bash
# 检查 BCE 服务状态
curl http://localhost:3000/health

# 检查参数格式
console.log(JSON.stringify(updateProgress));
```

---

### Q3: 无法按时完成怎么办？

**申请延期：**

```javascript
const extendDeadline = {
  operator: '匠心',
  reason: '技术难度超出预期，需要额外 1 天时间。',
  newDeadline: '2026-03-25T17:00:00Z'
};

const response = await fetch('http://localhost:3000/api/bce/manager/task-123/deadline', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify(extendDeadline)
});
```

**注意：** 延期需要管理者批准。

---

### Q4: 任务太复杂，无法独立完成怎么办？

**申请转交任务：**

```javascript
const reassign = {
  operator: '匠心',
  action: 'reassign',
  reason: '此任务需要前端开发经验，建议转交给灵犀。',
  suggestedAssignee: '灵犀'
};

const response = await fetch('http://localhost:3000/api/bce/tasks/task-123/confirm', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify(reassign)
});
```

**注意：** 转交需要管理者批准，且会扣 15 分绩效。

## 📝 附录

### 完整代码示例

**匠心机器人完整任务执行代码：**

```javascript
// 匠心机器人 - 任务执行模块
class TaskExecutor {
  constructor() {
    this.apiUrl = 'http://localhost:3000/api/bce/tasks';
    this.operator = '匠心';
  }
  
  /**
   * 确认任务
   */
  async confirmTask(taskId, comment = '任务已确认，立即开始执行。') {
    const confirm = {
      operator: this.operator,
      comment
    };
    
    const response = await fetch(`${this.apiUrl}/${taskId}/confirm`, {
      method: 'POST',
      headers: {'Content-Type': 'application/json'},
      body: JSON.stringify(confirm)
    });
    
    const result = await response.json();
    console.log('✅ 任务确认成功:', result);
    return result;
  }
  
  /**
   * 更新进度
   */
  async updateProgress(taskId, percent, completed, remaining, estimate, blockers = '无') {
    const progress = {
      percent,
      completed,
      remaining,
      estimate,
      blockers,
      operator: this.operator
    };
    
    const response = await fetch(`http://localhost:3000/api/bce/progress/${taskId}`, {
      method: 'POST',
      headers: {'Content-Type': 'application/json'},
      body: JSON.stringify(progress)
    });
    
    const result = await response.json();
    console.log('✅ 进度更新成功:', result);
    return result;
  }
  
  /**
   * 提交验收
   */
  async submitAcceptance(taskId, deliverables, selfTestReport, comment = '开发完成，请验收。') {
    const submit = {
      operator: this.operator,
      deliverables,
      selfTestReport,
      comment
    };
    
    const response = await fetch(`${this.apiUrl}/${taskId}/submit`, {
      method: 'POST',
      headers: {'Content-Type': 'application/json'},
      body: JSON.stringify(submit)
    });
    
    const result = await response.json();
    console.log('✅ 验收提交成功:', result);
    return result;
  }
}

// 使用示例
const executor = new TaskExecutor();

// 确认任务
executor.confirmTask('task-123');

// 更新进度
executor.updateProgress(
  'task-123',
  50,
  '三维分析模块、数据预处理',
  '建议生成模块、集成测试',
  '2026-03-24T17:00:00Z'
);

// 提交验收
executor.submitAcceptance(
  'task-123',
  ['智能决策模块代码', '三维分析算法文档', 'API 接口文档', '单元测试报告'],
  '自测报告：功能测试通过，性能测试通过，边界测试通过'
);
```
