## 🚨 三、故障处理

### 3.1 服务宕机

**症状：** `pm2 list` 显示 `errored` 或 `stopped`

**处理步骤：**
```bash
# 1. 查看错误日志
pm2 logs bce-server --lines 50

# 2. 重启服务
pm2 restart bce-server

# 3. 验证服务
curl http://localhost:3000/health
```

---

### 3.2 心跳任务停止

**症状：** 执行者反馈收不到任务提醒

**处理步骤：**
```bash
# 1. 检查心跳进程
ps aux | grep heartbeat-worker

# 2. 如果没有运行，重启
cd /Users/ai/.openclaw/workspace-jiangxin/匠心工作区/代码仓库/BCE-Project
nohup node scripts/heartbeat-worker.js > logs/heartbeat.log 2>&1 &

# 3. 验证
tail -20 logs/heartbeat.log
```

---

### 3.3 数据文件损坏

**症状：** 服务启动失败，报错 JSON 解析错误

**处理步骤：**
```bash
# 1. 备份当前数据
cd /Users/ai/.openclaw/workspace-jiangxin/匠心工作区/代码仓库/BCE-Project
cp runtime/bce-data.json runtime/bce-data.json.corrupted

# 2. 恢复备份
cp runtime/bce-data.json.backup.最新时间 runtime/bce-data.json

# 3. 重启服务
pm2 restart bce-server
```

---

### 3.4 磁盘空间不足

**检查磁盘空间：**

df -h /Users/ai/.openclaw/workspace-jiangxin/匠心工作区/代码仓库/BCE-Project
