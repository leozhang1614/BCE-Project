### 4. 定时轮询（BCE 辅助）

```javascript
// BCE 每 5 分钟扫描一次
cron.schedule('*/5 * * * *', async () => {
  // 扫描未确认任务
  const unconfirmedTasks = await getUnconfirmedTasks('匠心');
  
  // 如果有新任务，发送飞书提醒
  for (const task of unconfirmedTasks) {
    if (isNewTask(task)) {
      await sendFeishuReminder(task, '匠心');
    }
  }
});
```
