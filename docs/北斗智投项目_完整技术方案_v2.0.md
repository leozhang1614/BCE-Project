## 1. 项目概述

### 1.1 项目背景

**项目名称：** 北斗智投（Beidou Smart Investment）

**项目定位：**
- 智能投资决策支持系统
- 基于大数据的量化分析平台
- 个人投资者的智能投顾助手

**核心价值：**
- 📊 数据采集自动化 - 告别手动收集数据
- 🤖 分析建议智能化 - AI 驱动的投资建议
- 📈 决策支持可视化 - 直观的数据看板
- ⚠️ 风险预警实时化 - 及时的风险提示

### 1.2 项目范围

**包含内容：**
- ✅ 多源数据采集（财经网站、API、RSS）
- ✅ 数据清洗与存储
- ✅ 技术指标计算
- ✅ 投资建议生成
- ✅ Web 可视化看板
- ✅ 飞书消息推送

**不包含内容：**
- ❌ 实际交易执行（仅建议）
- ❌ 资金管理（用户自行决策）
- ❌ 合规咨询（需持牌机构）

### 4.3 分析引擎模块

**功能描述：**
- 基于规则的投资建议生成
- 多因子评分模型
- 风险等级评估

**分析维度：**

1. **技术面分析：**
   - 趋势判断（上涨/下跌/震荡）
   - 支撑位/压力位
   - 买卖信号识别

2. **基本面分析：**
   - 估值水平（PE、PB、PS）
   - 盈利能力（ROE、毛利率）
   - 成长性（营收增长、利润增长）

3. **资金面分析：**
   - 北向资金流向
   - 主力资金流向
   - 成交量变化

4. **消息面分析：**
   - 政策利好/利空
   - 行业动态
   - 公司公告

**评分模型：**
```
综合评分 = 技术面评分 × 30% + 基本面评分 × 40% + 资金面评分 × 20% + 消息面评分 × 10%

评级：
- 90-100 分：强烈推荐（⭐⭐⭐⭐⭐）
- 80-89 分：推荐（⭐⭐⭐⭐）
- 70-79 分：中性（⭐⭐⭐）
- 60-69 分：谨慎（⭐⭐）
- <60 分：回避（⭐）
```

**接口地址：**
```
http://hq.sinajs.cn/list=<股票代码>
```

**示例：**
```python
import requests

def get_stock_price(symbol):
    url = f"http://hq.sinajs.cn/list={symbol}"
    response = requests.get(url)
    data = response.text
    # 解析返回数据
    return parse_sina_data(data)

```

### 5.2 东方财富 API

**接口地址：**
```
https://push2.eastmoney.com/api/qt/stock/get
```

**参数：**
- `secid`: 市场代码 + 股票代码
- `fields`: 需要的字段

**示例：**
```python
def get_stock_detail(symbol):
    url = "https://push2.eastmoney.com/api/qt/stock/get"
    params = {
        "secid": f"1.{symbol}" if symbol.startswith("6") else f"0.{symbol}",
        "fields": "f43,f44,f45,f46,f47,f169,f170,f152,f133,f134,f135,f136,f137,f138,f139,f140,f141,f142,f143,f144,f145,f146,f147,f148,f149,f150,f151,f152,f153,f154,f155,f156,f157,f158,f159,f160,f161,f162,f163,f164,f165,f166,f167,f168,f169,f170"
    }
    response = requests.get(url, params=params)
    return response.json()
```

### 5.3 RSS 订阅

**支持的 RSS 源：**
- 雪球：https://xueqiu.com/rss
- 华尔街见闻：https://wallstreetcn.com/rss
- 财联社：https://cls.cn/rss

**示例：**
```python
import feedparser

def get_rss_news(url):
    feed = feedparser.parse(url)
    news_list = []
    for entry in feed.entries:
        news_list.append({
            "title": entry.title,
            "link": entry.link,
            "published": entry.published,
            "summary": entry.summary
        })
    return news_list
```
