## 5. 记忆刷新机制

### 5.1 刷新触发器

| 触发器 | 频率 | 说明 |
|--------|------|------|
| 定时刷新 | 每日 22:00 | 强制刷新当日记忆 |
| 心跳刷新 | 每次心跳 | Heartbeat 触发 |
| 事件刷新 | 实时 | 重大事件发生 |
| 会话结束 | 每次会话 | 会话历史归档 |

### 5.2 定时刷新流程

```
22:00 定时任务触发
    ↓
读取今日会话历史
    ↓
提取关键事件（决策/任务/问题）
    ↓
生成记忆草稿
    ↓
格式化写入 memory/YYYY-MM-DD.md
    ↓
Git 提交（自动 commit）
    ↓
检查 MEMORY.md 是否需要更新
    ↓
发送刷新完成通知（飞书）
    ↓
完成
```

### 5.3 刷新内容规则

**必须记录的事件：**
1. ✅ 磊哥决策/指令
2. ✅ 任务分配/变更
3. ✅ 技术方案批准
4. ✅ 问题与解决方案
5. ✅ 制度/规范变更
6. ✅ 人员角色调整

**可选记录的事件：**
1. 📝 日常对话摘要
2. 📝 技术讨论细节
3. 📝 非关键信息

**不记录的内容：**
1. ❌ 私人对话
2. ❌ 敏感信息（密码/密钥）
3. ❌ 临时性信息

### 5.4 刷新代码示例

```javascript
const cron = require('node-cron');
const fs = require('fs');
const path = require('path');

// 每日 22:00 自动刷新记忆
cron.schedule('0 22 * * *', async () => {
  console.log('[记忆刷新] 开始每日记忆刷新...');
  
  try {
    // 1. 获取今日会话历史
    const sessions = await getTodaySessions();
    
    // 2. 提取关键事件
    const events = extractKeyEvents(sessions);
    
    // 3. 生成记忆内容

    // 6. 发送通知

  } catch (error) {
    console.error('[记忆刷新] 失败:', error);
    await sendFeishuAlert('记忆刷新失败', error.message);
  }
});
```

## 6. 记忆断层修复

### 6.1 断层检测

**检测逻辑：**
```javascript
async function detectMemoryGap() {
  const today = new Date();
  const gaps = [];
  
  // 检查过去 7 天的记忆文件
  for (let i = 1; i <= 7; i++) {
    const date = new Date(today);
    date.setDate(date.getDate() - i);
    const dateStr = date.toISOString().split('T')[0];
    const filePath = path.join(__dirname, `memory/${dateStr}.md`);
    
    if (!fs.existsSync(filePath)) {
      gaps.push(dateStr);
    }
  }
  
  if (gaps.length > 0) {
    return {
      hasGap: true,
      missingDates: gaps,
      severity: gaps.length >= 3 ? 'HIGH' : 'MEDIUM'
    };
  }
  
  return { hasGap: false };
}
```

**检测频率：**
- 每日 08:00 自动检测
- 心跳时检测
- 手动触发检测

### 6.2 断层修复流程

```
检测到记忆断层
    ↓
分析缺失日期范围
    ↓
从会话历史提取数据
    ↓
生成记忆草稿
    ↓
人工确认（飞书通知）
    ↓
写入记忆文件
    ↓
Git 提交
    ↓
发送修复完成通知
    ↓
记录修复日志
    ↓
完成
```

### 6.3 修复代码示例

```javascript
async function repairMemoryGap(missingDates) {
  console.log('[记忆修复] 开始修复缺失记忆:', missingDates);
  
  for (const dateStr of missingDates) {
    try {
      // 1. 获取该日期的会话历史
      const sessions = await getSessionsByDate(dateStr);
      
      // 2. 提取关键事件
      const events = extractKeyEvents(sessions);
      
      // 3. 生成记忆内容
      const memoryContent = generateMemoryMarkdown(events);
      
      // 4. 写入文件
      const filePath = path.join(__dirname, `memory/${dateStr}.md`);
      fs.writeFileSync(filePath, memoryContent);
      
      // 5. 记录修复日志
      await logRepairAction(dateStr, 'AUTO_REPAIR');
      
      console.log(`[记忆修复] 完成：${dateStr}`);
    } catch (error) {
      console.error(`[记忆修复] 失败：${dateStr}`, error);
      await logRepairAction(dateStr, 'FAILED', error.message);
    }
  }
  
  // 6. Git 提交
  await gitCommit(`Repair memory gap: ${missingDates.join(', ')}`);
  
  // 7. 发送通知
  await sendFeishuNotification(
    '记忆修复完成',
    `已修复缺失记忆：${missingDates.join(', ')}`
  );
}
```
