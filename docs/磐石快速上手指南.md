# 磐石快速上手指南（10 分钟版）

**目标：** 10 分钟内学会 BCE 系统基础运维

---

## 第 1 分钟：认识系统

**BCE 系统是什么？**
- 北斗协同引擎（任务管理系统）
- 管理任务创建 → 执行 → 验收 → 审核全流程
- 通过飞书通知团队成员

**系统组成：**
1. **BCE 服务** - 主服务（PM2 管理）
2. **心跳任务** - 每 5 分钟查询待办任务
3. **健康监控** - 自动监控服务状态

---

## 第 2 分钟：服务状态检查

**打开终端，运行：**
```bash
pm2 list | grep bce
```

**正常输出：**
```
│ 0 │ bce-server │ online │
```

**异常输出：**
```
│ 0 │ bce-server │ errored │  ← 需要重启
│ 0 │ bce-server │ stopped │  ← 需要启动
```

---

## 第 3 分钟：服务重启

**最常用的命令：**
```bash
pm2 restart bce-server
```

**验证服务正常：**
```bash
curl http://localhost:3000/health
```

**看到 `{"status":"ok"}` 就是正常的！**

---

## 第 4 分钟：心跳任务检查

**心跳任务是什么？**
- 每 5 分钟主动查询待办任务
- 确保执行者能及时收到任务
- 不依赖飞书通知（兜底方案）

**检查心跳任务：**
```bash
ps aux | grep heartbeat-worker
```

**有输出就是正常的！**
