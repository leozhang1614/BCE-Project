## 📋 核心目标

**强制每个人每 5 分钟主动查询 BCE 任务，培养主动查询习惯。**

---

## 🎯 设计理念

### ❌ 错误方式（被动提醒）
```
定时任务发消息提醒 → 大家被动接收 → 可以忽略
```

### ✅ 正确方式（强制检查）
```
定时任务检查打卡记录 → 发现超时未查 → 上报天枢 + 磊哥
```

**关键区别：**
- 不是"提醒大家查"
- 而是"检查谁没查"
- 不查就会被发现，形成强制约束

## 🔄 工作流程

```
┌──────────────────────────────────────────────────────────────┐
│                     每个人的 Agent                            │
│                                                              │
│  ┌────────────────────────────────────────────────────────┐ │
│  │  定时任务 (每 5 分钟)                                     │ │
│  │  */5 * * * * python3 bce-checkin.py                    │ │
│  │                        ↓                                │ │
│  │  1. 调用 BCE API 查询"我的待办任务"                       │ │
│  │  2. 记录打卡时间到 last_check_in.json                   │ │
│  │  3. 发现待确认任务 → 自动确认                           │ │
│  │  4. 记录日志                                             │ │
│  └────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────┘
                            ↓
┌──────────────────────────────────────────────────────────────┐
│                    监督脚本 (每 5 分钟)                        │
│                                                              │
│  ┌────────────────────────────────────────────────────────┐ │
│  │  bce-supervisor.py                                      │ │
│  │                        ↓                                │ │
│  │  1. 检查所有人的 last_check_in.json                     │ │
│  │  2. 计算距离上次打卡的时间                               │ │
│  │  3. 超过 5 分钟未打卡 → ❌ 上报天枢 + 磊哥               │ │
│  │  4. 记录监督日志                                         │ │
│  └────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────┘
```

## 📊 具体实现

### 1. **打卡脚本 (bce-checkin.py)**

**位置：** `/Users/ai/.openclaw/agents/{agent}/scripts/bce-checkin.py`

**定时任务：**
```bash
*/5 * * * * python3 /Users/ai/.openclaw/agents/{agent}/scripts/bce-checkin.py
```

**脚本逻辑：**
```python
1. 调用 BCE API: GET /api/bce/tasks?assignee={名字}&status=pending
2. 记录打卡时间：~/.openclaw/agents/{agent}/last_check_in.json
3. 发现待确认任务 → 显示任务列表（不自动确认！）
4. 写入日志：~/.openclaw/logs/{agent}-bce-checkin.log
5. 提示成员前往 BCE 系统手动确认
```

**重要原则：**
- ❌ **不自动确认** - 让成员自己确认，培养主动意识
- ✅ **只显示待办** - 让成员知道自己有待办任务
- ✅ **提示确认时限** - 显示剩余时间和超时状态
- ✅ **提供确认链接** - 引导成员前往 BCE 系统确认

**打卡文件示例：**
```json
{
  "agent": "jiangxin",
  "name": "匠心",
  "last_check_in": "2026-03-22T14:23:00",
  "timestamp": 1774160580
}
```

### 2. **监督脚本 (bce-supervisor.py)**

**位置：** `/Users/ai/.openclaw/agents/jiangxin/scripts/bce-supervisor.py`

**定时任务：**
```bash
*/5 * * * * python3 /Users/ai/.openclaw/agents/jiangxin/scripts/bce-supervisor.py
```

**脚本逻辑：**
```python
1. 遍历 7 个人的 last_check_in.json
2. 计算：当前时间 - 上次打卡时间
3. 判断：
   - 如果 > 5 分钟：❌ 违规，上报天枢 + 磊哥
   - 如果 ≤ 5 分钟：✅ 正常
4. 写入监督日志
```

**监督逻辑：**
```python
CHECK_IN_INTERVAL = 5 * 60  # 5 分钟

for user in USERS:
    last_check_in = 读取打卡文件
    elapsed = 现在时间 - last_check_in
    
    if elapsed > 5 分钟:
        ❌ 上报：{user.name} 已 {elapsed} 分钟未查询任务
    else:
        ✅ 正常：{user.name} {elapsed} 分钟前已查询
```

## 🚨 异常处理

### 场景 1：脚本执行失败

**现象：** 打卡记录未更新

**排查：**
```bash
# 检查脚本是否存在
ls -la ~/.openclaw/agents/{agent}/scripts/bce-checkin.py

# 手动执行脚本
python3 ~/.openclaw/agents/{agent}/scripts/bce-checkin.py

# 查看错误日志
tail -n 50 ~/.openclaw/logs/{agent}-bce-checkin.log
```

### 场景 2：BCE 服务不可用

**现象：** 脚本报错 "连接失败"

**排查：**
```bash
# 检查 BCE 服务
ps aux | grep "node.*index.js"

# 测试 BCE API
curl http://localhost:3000/api/bce/tasks

# 重启 BCE 服务
pkill -f "node.*index.js" && cd ~/BCE-Project && node src/index.js &
```

### 场景 3：定时任务未执行

**现象：** 打卡记录长时间未更新

**排查：**
```bash
# 检查 crontab 配置
crontab -l | grep bce-checkin

# 检查 cron 服务
sudo systemctl status cron  # Linux
sudo launchctl list | grep cron  # macOS

# 重新配置定时任务
(crontab -l; echo "*/5 * * * * python3 ...") | crontab -
```

## 📝 使用说明

### 新人入职配置

1. **创建打卡脚本**
```bash
# 复制模板
cp ~/.openclaw/agents/jiangxin/scripts/bce-checkin.py \
   ~/.openclaw/agents/{new_agent}/scripts/bce-checkin.py

# 修改脚本中的 AGENT_NAME 和 CN_NAME
vi ~/.openclaw/agents/{new_agent}/scripts/bce-checkin.py
```

2. **添加定时任务**
```bash
(crontab -l; echo "*/5 * * * * python3 ~/.openclaw/agents/{new_agent}/scripts/bce-checkin.py") | crontab -
```

3. **更新监督脚本**
```bash
# 编辑 bce-supervisor.py，添加新人到 USERS 列表
vi ~/.openclaw/agents/jiangxin/scripts/bce-supervisor.py
```
