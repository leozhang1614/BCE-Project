🚀 BCE

北斗协同引擎 (Beidou Collaboration Engine)

✅ 服务状态：运行中

服务地址：http://localhost:3000

版本：v1.0.0 | 更新时间：2026-03-19

📡 可用 API 端点

  • GET /health 健康检查
    检查服务运行状态
  • GET / 根路径
    获取服务信息
  • POST /api/board 任务板
    创建/查询统一任务板（所有 Agent 共享）
  • POST /api/broadcast 消息广播
    发送全员广播消息，支持强制确认
  • POST /api/confirmation 强制确认
    创建消息确认请求，超时自动提醒
  • POST /api/memory 记忆桥接
    双写同步记忆（本地 + 中央库）
  • POST /api/feishu/webhook 飞书集成
    接收飞书群聊消息，自动转发到 BCE
  • GET /api/tasks 任务管理
    任务 CRUD 操作
  • GET /api/agents Agent 管理
    Agent 注册和状态查询

📖 快速开始

# 创建任务板
curl -X POST http://localhost:3000/api/board \
  -H "Content-Type: application/json" \
  -d '{"name": "测试任务板", "members": ["天枢", "匠心"]}'
# 发送广播
curl -X POST http://localhost:3000/api/broadcast \
  -H "Content-Type: application/json" \
  -d '{"title": "通知", "content": "明日 9:00 例会", "sender": "天枢", "recipients": ["匠心", "司库"], "requireConfirm": true}'
